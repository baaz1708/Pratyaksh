# Pratyaksh – Portfolio

A modern, single-page portfolio for Pratyaksh Singh highlighting education, experience, projects, skills, and activities.

## Getting started
1. Open `index.html` in your browser (or serve the folder with `python -m http.server 8000`).
2. Replace the placeholder photo at `assets/profile-placeholder.svg` with your professional picture to update the hero section.

No build steps or external tooling required.
